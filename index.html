# Основные команды Git **git config --global user.name 'User Name'** - добавить пользователя **git config --global user.email 'EMAIL'** - добавить email пользователя **git init** - назначить папку репозитарием
**git add** - добавить файл для отслеживания
**git commit -a -m** '*commment text*' - добавить комментарий к изменениям, где -a добавить файл на отслеживание, -m добавить комментарий
**git diff** - вывести изменения в файле
**git log** - вывести логи изменений файла **git branch name_branch** - создать ветку
**git checkout name_branch** - перейти в ветку
**git status** - состояние рабочего каталога и проиндексированных файлов
**git merge** - слить ветки **git clone** - сделать локальный клон удаленного репозитария **git push** - скачать ветку с удаленного репозитария **git pull** - отправить ветку на удаленный репозитарий
